# Templates (esqueletos opcionais)

Modelos de partida que o `ca-redator` pode copiar e preencher. Não são
obrigatórios — servem para acelerar e padronizar. Cada `<slug>` de material vive
em `materiais/<slug>/`; estes arquivos são só ponto de partida do `material.md`.

- `plano-de-aula.md` — plano formal de aula.
- `roteiro-de-aula.md` — passo a passo com tempos.
- `apostila.md` — estrutura de apostila/e-book.
