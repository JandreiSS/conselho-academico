# Tipos de material didático

Guia de referência para o Coordenador reconhecer o que cada material exige e
quais etapas do conselho fazem mais sentido. Leia a seção do tipo pedido.

## Roteiro de aula

Passo a passo do que o professor faz e fala durante a aula, com tempos.
- **Estrutura:** abertura/acolhida → problematização → desenvolvimento (momentos)
  → prática → fechamento/síntese → tarefa.
- **Marcações de tempo** por momento (ex.: 0–5 min, 5–20 min...).
- Falas-chave, perguntas disparadoras, o que escrever no quadro/slide.
- **Etapas:** designer (objetivos + sequência) → pesquisa (leve) → redação →
  avaliação (opcional) → revisão.

## Plano de aula

Documento formal de planejamento (mais enxuto que o roteiro).
- **Campos:** identificação (disciplina, ano, duração), objetivos, conteúdos,
  metodologia, recursos, avaliação, referências.
- Alinhamento a competências/habilidades (BNCC quando aplicável).
- **Etapas:** designer (peso maior) → redação → revisão.

## Sequência didática

Conjunto encadeado de aulas/atividades sobre um tema, progressivo.
- Visão geral + cada aula com objetivo, atividades e avaliação.
- Progressão de complexidade e retomada.
- **Etapas:** designer (sequência) → pesquisa → redação → avaliação → revisão.

## Apostila

Material de estudo do aluno, com teoria + exemplos + exercícios.
- **Estrutura:** sumário → introdução → capítulos/tópicos (teoria, exemplos
  resolvidos, boxes) → exercícios por tópico → gabarito → referências.
- Linguagem didática, exemplos progressivos, destaques visuais.
- **Etapas:** todas — é o caso completo.

## Capítulo de livro / livro didático

Texto autoral aprofundado, com rigor e progressão.
- **Estrutura:** abertura (o que e por quê) → seções conceituais → exemplos e
  estudos de caso → síntese → atividades → leituras complementares.
- Cuidado com precisão conceitual e citação de fontes.
- **Etapas:** todas, com pesquisa forte.

## E-book

Como o livro, porém mais visual e modular, pensado para leitura em tela.
- Capítulos curtos, chamadas, resumos, elementos gráficos.
- **Saída** frequentemente PDF/HTML; considere material visual (capa) via editor.

## Lista de exercícios / avaliação

Foco em questões e correção.
- Questões variadas alinhadas a objetivos, gabarito e rubricas.
- **Etapas:** designer (objetivos) → avaliador (peso maior) → revisão.

---

**Regra prática:** o tipo define quais profissionais têm peso maior. Sempre
confirme o objetivo pedagógico — é ele que orienta todas as etapas.
